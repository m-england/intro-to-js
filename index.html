Introduce yourself

What is it?

JavaScript is a programming language that confusingly has nothing to do with Java. JavaScript is a cross-platform, object-oriented scripting language When JavaScript was created, it initially had another name: “LiveScript”. But Java was very popular at that time, so it was decided that positioning a new language as a “younger brother” of Java would help.
And now...
World's most popular programming language
It's ubiquitous, you probably use it everyday on every device you have. Why?
Programming language of the Web
(Boring preface) Created in Summer `95 by Brendan Eich working for Netscape.
But as it evolved, JavaScript became a fully independent language with its own specification called ECMAScript, and now it has no relation to Java at all.
You can find it in almost every webpage, but it is also used for server applications with node, and electron (mention things like slack and discord).

Statements

All JS programs are comprised of a list of "statements" that execute one by one in the order they are written.

Statements


                let name, age;
                name = 'Mark';
                age = 35;
                let gameOver = age + 50;

Variables


                let name, age;
                name = 'Mark';
                age = 35;
                let gameOver = age + 50;
JavaScript variables are containers for storing data values. In this example: name, age and gameOver are variables. declared with the let keyword.
Variables are assigned values with an equals sign. For use in other statements.

Types


                let name = 'Mark';
                let age = 35;
                let isWearingShoes = true;
JavaScript is a dynamically typed language. That means that variables can change type from one to another. It also means that javascript can mix types.

Objects


                let person = {
                    name: 'Mark';
                    age: 35,
                    isWearingShoes: true
                }

                if (person.isWearingShoes) {
                    console.log(`Hey ${person.name}, nice shoes!`)
                } else {
                    console.log(`Hey ${person.name}, this is a Walmart.`)
                }
A JavaScript object has properties associated with it. A property of an object can be explained as a variable that is attached to the object. Object properties are basically the same as ordinary JavaScript variables, except for the attachment to objects. The properties of an object define the characteristics of the object.
