Statements

All JS programs are comprised of a list of "statements" that execute one by one in the order they are written.

Statements


                let name, age;
                name = 'Mark';
                age = 35;
                let gameOver = age + 50;

Variables


                let name, age;
                name = 'Mark';
                age = 35;
                let gameOver = age + 50;
JavaScript variables are containers for storing data values. In this example: name, age and gameOver are variables. declared with the let keyword.
Variables are assigned values with an equals sign. For use in other statements.

Functions

Browser functionality

Web browsers (chrome, edge, firefox) all have functionality that they make available to JS code. The functions are made available as "Browser APIs". Anything that a webpage does that affects your computer is accomplished through Browser APIs.
