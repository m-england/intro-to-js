What is it?

JavaScript is a cross-platform, object-oriented scripting language When JavaScript was created, it initially had another name: “LiveScript”. (Boring preface) Created in Summer `95 by Brendan Eich working for Netscape.
Java was very popular at that time, so it was decided that positioning a new language as a “younger brother” of Java would help. It evolved, JavaScript became a fully independent language with its own specification called ECMAScript, and now it has no relation to Java at all.
And now...
World's most popular programming language
It's ubiquitous, you probably use it everyday on every device you have. Why?
Programming language of the Web
You can find it in almost every webpage, but it is also used for server applications with node, and electron (mention things like slack and discord).

Statements

All JS programs are comprised of a list of "statements" that execute one by one in the order they are written.

Statements


                let name, age;
                name = 'Mark';
                age = 35;
                let gameOver = age + 50;

Variables


                let name, age;
                name = 'Mark';
                age = 35;
                let gameOver = age + 50;
JavaScript variables are containers for storing data values. In this example: name, age and gameOver are variables. declared with the let keyword.
Variables are assigned values with an equals sign. For use in other statements.

Types


                let name = 'Mark';
                let age = 35;
                let isWearingShoes = true;
JavaScript is a dynamically typed language. That means that variables can change type from one to another. It also means that javascript can mix types.

Objects


                let person = {

                if (person.isWearingShoes) {

A JavaScript object has properties associated with it. A property of an object can be explained as a variable that is attached to the object. Object properties are basically the same as ordinary JavaScript variables, except for the attachment to objects. The properties of an object define the characteristics of the object. Requests

The Web is built on APIs (Application Programming Interface). Essentially, a WebApi is a computer program on the network that is making its data or services available to be contacted. In JS land, these APIs usually transfer data in a format called JSON, and JS makes network requests and receives network responses with a built-in object called XMLHttpRequest.

What's next for JavaScript?

The ECMAScript is still evolving but has certainly stabilized relative to its former self.
Languages are built around it like typescript and coffeescript, which aim to shore up the shortcomings of JS.
Browsers are better and faster at compiling JS than ever before and have come a long way in the last few years.
WebAssembly is the looming "boogeyman" of the JS world.
